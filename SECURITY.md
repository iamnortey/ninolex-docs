# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Ninolex or referenced in this documentation, please report it responsibly.

**Email:** [isaac@nortey.dev](mailto:isaac@nortey.dev)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy covers the public documentation. For vulnerabilities in the Ninolex pronunciation engine, report to the code repositories directly. For the open-source dictionary, report at [ninolex-gh](https://github.com/iamnortey/ninolex-gh).
